Installing the GeoServer OGC API moduleÂ¶

  1. Download the OGC API nightly GeoServer community module from geoserver-2.22.4-ogcapi-plugin.zip.

    Warning

    Verify that the version number in the filename corresponds to the version of GeoServer you are running (for example geoserver-2.22.4-ogcapi-plugin.zip above).

  2. Extract the contents of the archive into the WEB-INF/lib directory of the GeoServer installation.

  3. On restart the services are listed at http://localhost:8080/geoserver
